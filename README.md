# Automatic Hand Sanitizer 🧴

IoT based touchless sanitization system using Arduino UNO.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- 5V Power Supply
- Jumper Wires

## How It Works
Ultrasonic sensor detects hand within 10cm range,
Arduino triggers servo motor to dispense sanitizer automatically.

## Tech Stack
- Embedded C
- Arduino IDE

## Features
- Touchless operation
- Auto dispensing within 10cm range
- Low power consumption
